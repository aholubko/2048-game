# 2048 Game
A browser-based implementation of the classic 2048 puzzle game built with JavaScript.

The goal of the game is to combine tiles with the same value until a tile with the number 2048 is created. The project demonstrates object-oriented programming, DOM manipulation, keyboard event handling, and game-state management.

## Live Preview
[Play the game](https://aholubko.github.io/2048-game/)

- Git
- GitHub

## Features
- Classic 4 × 4 game board
- Tile movement using keyboard arrow keys
- Merging of tiles with equal values
- Automatic generation of new tiles after each valid move
- Score calculation based on merged tiles
- Win state when the 2048 tile is created
- Game-over detection when no moves are available
- Start and restart functionality
- Responsive interface

## How to Play
1. Click the **Start** button.
2. Use the keyboard arrow keys to move the tiles:
   - `Arrow Up`
   - `Arrow Down`
   - `Arrow Left`
   - `Arrow Right`
3. Tiles with equal values merge when they touch.
4. Continue combining tiles until you create the `2048` tile.
5. Use the **Restart** button to start a new game.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/aholubko/2048-game.git
cd 2048-game
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project locally

```bash
npm start
```
